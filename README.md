# AxoraLab

AxoraLab için Next.js App Router tabanlı pazarlama sitesi ve demo route'ları.

## Son güncelleme

Ana sayfa tamamen yeniden kuruldu.

Bu redesign ile:
- light section yapısı kaldırıldı
- ana sayfa dark-only sinematik product studio yönüne taşındı
- eski dar agency/finance odaklı homepage kurgusu temizlendi
- pricing, Vibe Coding ve sektör bazlı dar örnekler homepage'ten çıkarıldı
- hero alanı gerçek operasyon yazılım yüzeyleri hissi verecek şekilde yeniden tasarlandı
- studio showcase alanı anlamsız KPI/graph yerine workflow, operations console ve data sync yüzeyleriyle yeniden kuruldu
- nav ve footer karanlık sisteme göre sadeleştirildi
- demo route'ları ve genel uygulama yapısı korundu

## Geliştirme

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
